1) Se crea el archivo Dockerfile

2) Se crea el archivo run.sh

3) Se crea una red en docker manualmente
docker network create -d bridge --subnet 192.167.54.0/24 --gateway 192.167.54.1 net_terminal

3) Se crea el archivo docker-compose.yml

4) Se crea el archivo settings.env donde estar el usuario y pass, el archivo esta cifrado.
    USER=usuario
    PASS=password

NOTA: Dede de ser diferente para cada servidor.

5) Se crea el archivo up_terminal.sh

6) Se ejecuta el script y en el navegador poner:
    direccion_ip:6767

7) El usuario es limitado, se ejecutara el comando ssh
    ssh usuario@192.167.54.1

El admin debe de conocer las credenciales:
    1) credenciales para entrar a tty
    2) credenciales para entrar por ssh
