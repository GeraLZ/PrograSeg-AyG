Instalar:
    - pip
        - con pip instalar django (pip3 install django)
        - con pip instalar mysqlclient (pip3 install mysqlclient)
    - ccrypt

Para crear el proyecto:
    - django-admin startproyect monitor

Crear un nuevo archivo dentro de la carpeta que se creo al ejecutar el anterior comando.El arhivo se llamara settings.env
    - En mi caso ya el archivo se creo en:
        /home/adalucs/Documentos/PrograSegura/monitor/settings.env

En el archivo settings.py que se creo al crear el nuevo proyecto(monitor):
    - Quitar el valor de la variable SECRET_KEY y pasarlo al archivo de settings.env, de la sig manera
        SECRET_KEY='VALOR_SECRET_KEY'
    - Se cambiara el valor de SECRET_KEY de la siguiente manera:
        SECRET_KEY = os.environ.get('SECRET_KEY')
    - Se cifrara el archivo settings.env de la sig manera:
        ccencrypt settings.env
    NOTA: Se se quiere descifrar usar: ccdecrypt settings.env.cpt
    NOTA: Password == 980922 la pongo aqui porque luego se me olvida T_T

Crear un script en el mismo lugar que se creo el archivo settings.env este se llamara startServer.sh
    - Nos va a servir para exportar las variables de entorno e iniciar el servidor.
            NOTA: El script esta en otro archivo, pero esta bonito y le hice comentarios <3

Para crear una base de datos:
    - python manage.py startapp db
    - En la parte INSTALLED_APPS agregar el nombre de la app al final con comillas simples.
        'db',
    - En la parte DATABASES cambiar con lo siguiente:

        DATABASES = {
            'default': {
                'ENGINE': 'django.db.backends.mysql',
                'NAME': os.environ.get('DB_NAME'),
                'USER': os.environ.get('DB_USER'),
                'PASSWORD': os.environ.get('DB_PASSWORD'),
                'HOST': os.environ.get('DB_HOST'),
                'PORT': os.environ.get('DB_PORT'),
            }
        }

    - En el archivo de settings.env agregar lo siguiente abajo de SECRET_KEY
        DB_NAME=monitor
        DB_USER=admin
        DB_PASSWORD=monitor2020
        DB_HOST=localhost
        DB_PORT=3306

    - En mysql crear la base de datos y el usuario.
        create database monitor;
        grant all on monitor.* to 'admin'@'localhost' identified by 'monitor2020';

    - Se creara otro script que se encargara de hacer makemigrations y migrate.
